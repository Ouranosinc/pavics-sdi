Modify a provided PAVICS WPS workflow for a similar but different purpose¶

One potentially powerful tool for the Climate Scenarios and Services team is to provide users with example workflows that can then be modified to different needs¶

In this exercice we will ask you to modify the provided PAVICS workflow example below in order to : 1. Change the spatial subset to a region of your choice * Select a different set of 3 consecutive years (2008-2010) - 3 only to keep time and processing reasonable * Calculate a 2 additional climate indices using CDO - monmax (previous) as well as monmin and monmean * Download the resulting .nc files locally and make a plot - the plot should contain : original daily data, with monthly min, mean, and max

PAVICS WPS urls: * PAVICS catalogue : https://pavics.ouranos.ca/twitcher/ows/proxy/catalog/pywps * Flyingpigeon suite : https://pavics.ouranos.ca/twitcher/ows/proxy/flyingpigeon/wps * Hummingbird suite : https://pavics.ouranos.ca/twitcher/ows/proxy/hummingbird/wps
