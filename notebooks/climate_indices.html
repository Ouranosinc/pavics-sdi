Climate indicesÂ¶

The current production version of PAVICS provides ICCLIM indices through the FlyingPigeon server, which includes over fourty different indices. In the example below, weâ€™re calling on one of these indices on a small test file available on a public URL, but it is also possible to send local files to the server.

Another implementation of climate indices is in the works. It relies on the xarray+dask backend to parallelize computations in the background. The library of climate indices is xclim and the WPS server hosting it is finch, but is not yet in production.

In [1]:

%matplotlib inline
from birdy import WPSClient
url = 'https://pavics.ouranos.ca/twitcher/ows/proxy/flyingpigeon/wps'
fp = WPSClient(url)
